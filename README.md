This project encompasses an application for a hospital management system, including user (patient), doctor, and administrator workflow steps. Different workflow steps have been defined for users, doctors, and administrators.

**User (Patient) Workflow Steps:**
- Registration and login to access the personal panel.
- Appointment scheduling, viewing existing appointments, and canceling appointments.

**Doctor Workflow Steps:**
- Doctors log in to view their appointments.
- Ability to update profile information (specialization, working hours, etc.).

**Administrator Workflow Steps:**
- Administrator panel for appointment management (updating or deleting appointments).
- Management of staff (doctors) and departments (adding new doctors, editing departments).

**Technologies:**
-C#, .NET, 

**Project Objective:** This system aims to facilitate effective appointment management, profile updates, and department management between users (patients), doctors, and administrators in a hospital setting.
